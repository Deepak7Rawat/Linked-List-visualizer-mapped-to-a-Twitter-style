// About the project

From C++ to a beautiful browser experience.

A college DSA project reimagined as a portfolio-grade web app — every operation maps 1:1 to the underlying linked list code.

📚 Academic core

Built around a doubly & singly linked list — the same data structure your DSA syllabus loves.

🎨 Modern UI

Glassmorphism, gradients, soft shadows and animated nodes — designed to look great in your portfolio.

🔌 Backend ready

UI is decoupled from logic — drop in any storage layer, no UI rework needed.

// Features

Everything a linked list can do.

➕ Insert

At beginning, end, or any position — animated in real time.

🗑️ Delete

Remove from head, tail, or by index. Pointers re-wire automatically.

🔍 Search

Find by username or tweet ID with instant highlight feedback.

🔁 Traversal

Forward in singly mode, forward & backward in doubly mode.

🧠 Two modes

Switch between singly and doubly linked list with one click.

🐦 Live feed

Every node appears as a tweet card — auto-synced with the list.
